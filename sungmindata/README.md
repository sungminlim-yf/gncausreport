# sungmindata — 살아있는 사업 자료 (주제·독자 카드의 근거)

이 폴더는 **지엔씨에너지(GNC Energy)와 현재 추진 중인 호주 사업에 대한 1차 자료**를 올리는 곳입니다.
목적은 단 하나 — **보고 자동화가 "고정된 주제 반복"에 갇히지 않게** 하는 것입니다.

## 왜 필요한가

정기 보고는 `topics.md`의 주제 목록에 근거합니다. 이 목록이 갱신되지 않으면 **비슷한 보고가 반복**되어
실효성이 떨어집니다. 그래서 여기에 **살아있는 사업 자료**를 올리면, Claude가 이를 읽고
`/refresh-topics` 명령으로 **정말 우리에게 도움이 될 주제를 새로 도출·갱신**합니다.
(주제를 직접 손으로 적기보다, 자료에 근거한 판단으로 더 유익한 주제를 고르게 하는 구조)

## 무엇을 올리나

우리 사업의 *현재 상태와 의도*를 드러내는 자료일수록 좋습니다.

- **회사·제품**: GNC 회사 소개, 제품 카탈로그/사양(디젤·가스터빈 발전기 등), 레퍼런스 실적
- **사업 추진 현황**: 호주 진출 계획·타깃 시장·일정, 내부 검토 메모, 의사결정 맥락
- **협력사/고객 교신**: Eneraque 등 파트너와 주고받은 살아있는 자료, 고객·입찰 관련 메모(민감 정보 주의)
- **미해결 질문**: "이건 더 알아봐야 한다" 싶은 열린 이슈 목록

## 형식

- **권장**: `.md`, `.txt`, `.pdf` (Claude가 바로 읽음). 표·수치는 가능하면 텍스트로.
- **이미지**(`.png`/`.jpg`): 제품 사진·다이어그램도 읽을 수 있음.
- **.docx/.xlsx/.pptx**: 가능하면 PDF로 내보내거나 핵심을 `.md`로 붙여넣어 주세요(직접 파싱은 제한적).
- 파일명은 내용을 알 수 있게 (예: `2026-06_GNC회사소개.pdf`, `2026-06_Eneraque-MOU초안.md`).

## 어떻게 쓰이나 (워크플로)

1. 이 폴더에 자료를 올린다.
2. `/refresh-topics` 실행 → Claude가 **이 폴더 + 현행 `topics.md` + 기보고(`posted-index`) + 독자 카드**를 함께 읽고:
   - 이미 충분히 다룬 주제는 **반복 회피**(심화·후속으로 전환)
   - 의사결정에 필요한데 비어 있는 **정보 격차**를 새 주제로 제안
   - 각 제안이 *이 폴더의 어떤 자료*에서 비롯됐는지 **근거 표시**
   - 자료에서 드러난 우선순위·맥락으로 **독자 카드(audience) 갱신안**도 제안
3. **사람이 승인**하면 `topics.md`·`audience`에 반영(완전 자동 아님, D2 정신).
